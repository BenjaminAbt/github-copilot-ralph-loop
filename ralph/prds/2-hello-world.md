# User Story 2 - Hello World

Story: 002

## Description
As a user, I want the application to print a random greeting in different languages so that I can see a diverse Hello World output.

## Scope
- Implement greeting logic
- Maintain a fixed list of greetings and languages
- Random selection on each application start

## Acceptance Criteria
- Running the application prints exactly one greeting
- The output includes:
  - the greeting text
  - the corresponding language
- The greeting is selected from a predefined list
- The application can be executed using `dotnet run`
